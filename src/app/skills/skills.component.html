Swift

Swift is the main language of choice in modern macOS and iOS development. Objective-C is still the underlying language but, Swift is the main language that runs Apple's set of devices. Swift is a very nice language and its tools and built-in libraries are very well developed.

Typescript

Typescript is a language built on top of Javascript and it is the main language that runs Angular projects. Typescript is a very cool language and has great feature and libraries that integrate into Angular very well. Since Typescript is a Javascript language, NPM works directly with it, making library installations very easy to install and maintain.

Go

The Go programming language is known to be very efficient. It is, in my opinion, the best language to run a large web application on the back-end. It has feature like go routines, which are used in multi-threading and are extremely memory efficient. Go also has a very good CLI, making libraries very easy to install and use in your Go project.

Frameworks

Angular

Angular is my favorite web framework at the moment. It is an organized framework with components, services, classes, structs, and more. It is a very responsive framework especially with its custom Angular Material Css and Typescript library. Angular makes developing large scale web applications on a team very managable and efficient.

F3

F3 is a very powerful but light PHP back-end framework. It is, in my opinion, very easy to learn because of its consistency. F3 has very nice libraries built, for example, lots of different SQL libraries that are maintained and secured. It is, of course, completely open source and hightly active.

Laravel

The Laravel framework is developed object orientedly. It has a very nice CLI to manage your project and it scales fairly well across a team. Laravel is used by many companies and trusted by many companies because of its highly active rolling releases and developers.

Electron

ElectronJS is a framework built on Javascript to build desktop applications using web languages include HTML, CSS, and Javascript. Electron integrates with NPM and allows easy installations of all NPM libraries. It uses the Chromium engine to build desktop applications and has very talented developer on its team keeping Electron up to date.

JQuery

JQuery is an amazing Javascript library. It allows very easy dom selections and manipulation. JQuery gives developers the ability to highly condense their Javascript. JQuery is very highly used because of it time savings and scalability.
