The "Nick's Desk" philosophy.

A cool software developer's philosophy by Nick Gerancher

Hi, my name is Nicholas Gerancher. I am a software developer as well as a web developer, currently developing this portfolio as you are reading it! As a developer, I have many self-developed personal habbits and principles when developing projects. For example, when starting a project, I always end up using Git and sublime text (for most web projects and some C++ projects). Git is a vital tool for me as a developer as it allows very quick and seemless transition between work stations and developers.
